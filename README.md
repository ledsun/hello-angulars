# Hello Angulars

AngularJSとAngularの違いを検証します。
AngularJSとAngularを使った極力シンプルなアプリケーションで比較します。

https://angularjs.org/ の `Add Some Control` を題材にします。

## AngularJS

```sh
cd angular-js
python -m http.server 4200
```

http://localhost:4200 で起動します。

## Angular

```sh
cd angular-js
npm install
npm start
```

http://localhost:4200 で起動します。
